# Inventory Upgrade

![Where does it keep all this stuff...](oredict:oc:inventoryUpgrade)

The Inventory Upgrade provides inventory slots to [robots](robot.md) and [drones](drone.md). For each inventory upgrade a robot will gain an addition 16 inventory slots, up to a maximum of 64 slots in total, a drone will gain 4 slots, up to a maximum of 8 slots in total.

If no inventory upgrade is installed in a device it will not be able to store or pick up items.
